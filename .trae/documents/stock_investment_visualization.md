# 中国股市投资收益可视化网页 - 实施计划

## 一、Summary（摘要）

构建一个单页网页，可视化展示中国股市从 2024 年 7 月 19 日至今的模拟走势，以及一名大学生从该日起每月定投策略下的累计收益曲线。使用纯 HTML/CSS/JavaScript + Chart.js 实现，单文件部署，无需后端与构建工具。

## 二、Current State Analysis（现状分析）

- 工作目录 `/workspace` 仅包含一个 `README.md`（内容为 `g_python`），无任何前端代码。
- 需要从零搭建一个独立的静态网页项目。
- 无既有依赖、无构建配置、无样式约定，可自由设计。

## 三、Proposed Changes（拟定变更）

### 文件结构
```
/workspace/
├── README.md              （已存在，不修改）
├── index.html             （新建：页面骨架与样式）
├── app.js                 （新建：数据模拟 + 定投计算 + 图表渲染）
└── data.js                （新建：模拟数据生成与常量定义）
```

### 文件 1：`/workspace/index.html`
**做什么**：网页骨架、UI 布局、内嵌 CSS、引入 Chart.js CDN 与本地脚本。
**为什么**：承载页面结构、统计卡片、图表容器；单页应用入口。
**怎么做**：
- `<head>` 引入 Chart.js CDN：`https://cdn.jsdelivr.net/npm/chart.js@4.4.1/dist/chart.umd.min.js`
- 内嵌 CSS 样式（深色金融主题，红涨绿跌符合中国习惯）：
  - 顶部标题栏：标题 + 日期范围
  - 统计卡片区域：4 个卡片（累计投入、当前市值、累计收益、收益率%）
  - 主图表区：双 Y 轴折线图（大盘指数 + 投资收益曲线）
  - 副图表区：定投收益 vs 投入成本对比
- 页脚说明：数据为基于真实趋势的模拟数据，仅供参考
- `<body>` 末尾引入 `data.js` 和 `app.js`

### 文件 2：`/workspace/data.js`
**做什么**：生成从 2024/7/19 到今天（运行时动态计算）的模拟股市日数据。
**为什么**：用户选择"模拟数据（基于真实趋势）"方案，需在浏览器端按真实大盘规律生成可复现的走势。
**怎么做**：
- 起始基准点：参考上证指数 2024/7/19 收盘约 2982 点
- 使用**带种子的伪随机游走**（mulberry32 种子算法），保证每次刷新数据一致
- 仅生成交易日（跳过周末与中国法定节假日简易判断）
- 注入真实趋势特征：
  - 2024/7~9：底部震荡（小幅波动）
  - 2024/9~10：924 行情急涨（+20% 脉冲）
  - 2024/11~2025/1：高位回落震荡
  - 2025/2~2025/6：缓慢反弹
  - 2025/7~至今：震荡上行
- 输出结构：`{ date: 'YYYY-MM-DD', index: number }` 数组
- 导出全局变量 `MARKET_DATA`

### 文件 3：`/workspace/app.js`
**做什么**：定投计算逻辑 + 图表渲染 + 统计指标更新。
**为什么**：用户选择"每月定投"策略，需在市场数据基础上计算每月投入后的累计成本、市值、收益。
**怎么做**：
- **定投参数**：每月 19 日（若非交易日则顺延到下一交易日）投入 ¥1000
- **份额计算**：以当日模拟指数作为"基金净值"近似，份额 = 1000 / (index/1000)，即把指数除以 1000 作为单位净值
- **累计计算**：维护累计份额、累计成本；每个交易日按当日指数计算当前市值
- **输出序列**：
  - `investmentCurve`：每日当前市值
  - `costLine`：每日累计投入成本（阶梯线）
- **关键指标**：
  - 累计投入金额
  - 当前市值
  - 累计收益（市值 - 投入）
  - 收益率 % （收益 / 投入 × 100）
- **Chart.js 渲染**：
  - 主图：双 Y 轴折线图
    - 左 Y 轴：大盘指数（红色线）
    - 右 Y 轴：投资市值（蓝色线）
    - X 轴：日期
  - 副图：收益 vs 成本对比
    - 成本阶梯线（灰色）
    - 市值曲线（蓝色）
    - 收益填充区（红/绿）
- **交互**：鼠标悬浮显示当日详情 tooltip
- 入口：`document.addEventListener('DOMContentLoaded', init)`

## 四、Assumptions & Decisions（假设与决策）

1. **数据范围**：从 2024/7/19 到运行当天（动态计算 endDate = new Date()）。
2. **数据可复现**：使用固定种子随机数，保证每次打开页面数据走势一致。
3. **定投日**：每月 19 日（与起始日呼应），非交易日顺延。
4. **定投金额**：每月 ¥1000（大学生合理水平，可在代码顶部常量修改）。
5. **单位净值近似**：用 `指数/1000` 作为基金净值，方便份额计算（仅为可视化用途，非真实产品）。
6. **节假日处理**：仅跳过周末，不精确处理中国调休（简化实现，对长期趋势影响可忽略）。
7. **配色**：遵循中国股市习惯 —— 红涨绿跌；整体深色金融风。
8. **免责声明**：页面明确标注"数据为模拟数据，仅供参考，不构成投资建议"。
9. **不修改** `README.md`。
